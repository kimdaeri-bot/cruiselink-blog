# 크루즈 짐싸기 완벽 체크리스트: 뭘 가져가야 할까? 🧳

크루즈 여행이 확정되면 다음 고민은 **"뭘 가져가지?"**입니다. 일반 여행과 다르게, 크루즈에는 가져가야 할 것과 **가져가면 안 되는 것**이 따로 있습니다.

이 크루즈 짐싸기 체크리스트를 저장해두고 출발 전에 확인하세요!

---

## ✅ 필수 준비물 — 이건 꼭!

### 서류 📄
- [ ] **여권** (유효기간 6개월 이상 남아 있는지 확인!)
- [ ] **크루즈 예약 확인서** (인쇄 + 모바일)
- [ ] **항공권 E-티켓** (해외 출발항의 경우)
- [ ] **여행자 보험 증서**
- [ ] **비자** (필요한 경우 — 미국 ESTA, 쉥겐 등)
- [ ] **신용카드/체크카드** (해외 결제 가능한 것)

### 의류 👔
- [ ] **캐주얼 복장**: 낮 시간 선내 활동용 (반팔, 반바지 등)
- [ ] **저녁 복장**: 메인 다이닝용 (캐주얼 엘레강스 이상)
- [ ] **수영복**: 2벌 이상 (매일 사용하므로 교체용)
- [ ] **운동복/운동화**: 피트니스, 기항지 도보 관광
- [ ] **가벼운 겉옷**: 선내 에어컨이 강할 수 있음
- [ ] **포멀 복장**: 포멀 나이트가 있다면 (아래 드레스코드 참고)

### 세면/건강용품 🧴
- [ ] **선크림** SPF50+ (바다 위 자외선은 강력합니다)
- [ ] **선글라스**
- [ ] **상비약**: 소화제, 진통제, 밴드, 개인 처방약
- [ ] **멀미약**: 키미테, 보나링 또는 스코폴라민 패치 ([멀미 가이드](/cruiselink-blog/가이드/cruise-seasickness-guide/) 참고)
- [ ] **칫솔/치약**: 선내 제공 안 하는 선사 많음
- [ ] **세면용품**: 샴푸, 바디워시 (기본 제공하지만 소량)

### 전자기기 ⚡
- [ ] **멀티 어댑터/돼지코**: 선사별 콘센트 규격이 다름 (미국식/유럽식)
- [ ] **보조 배터리**: 기항지 관광 시 필수
- [ ] **USB 충전 케이블**: 여분 포함
- [ ] **카메라/액션캠**: 기항지 풍경 촬영용
- [ ] **방수 폰케이스**: 수영장, 해변용

## 💡 선택 준비물 — 있으면 좋은 것

### 선내 편의용품
- [ ] **자석 후크**: 크루즈 객실 벽은 철재! 자석으로 물건 걸기 가능 🧲
- [ ] **미니 전원 탭**: 객실 콘센트가 부족함 (멀티탭 ✅, 연장코드 ❌)
- [ ] **미니 가습기**: 선내가 건조할 수 있음
- [ ] **클립/집게**: 발코니에 수영복 말릴 때
- [ ] **접이식 가방**: 기항지 쇼핑용
- [ ] **이어플러그/수면 안대**: 숙면용
- [ ] **물병**: 기항지 외출 시 (선내에서 리필)

### 기항지 관광용
- [ ] **편한 워킹슈즈**: 유럽 기항지는 돌바닥, 걷기 많음
- [ ] **소형 백팩/크로스백**: 귀중품 보관
- [ ] **현금 (소액)**: 기항지 소규모 상점, 팁용
- [ ] **우비/접이식 우산**: 갑자기 비 올 수 있음

### 특수 상황
- [ ] **스노클링 장비**: 카리브해/동남아 (렌탈도 가능)
- [ ] **쌍안경**: 알래스카 고래 관찰, 빙하 감상
- [ ] **와인 오프너**: 일부 선사는 와인 반입 허용 (1병)

## 🚫 금지 품목 — 절대 안 돼!

크루즈선에 가져갈 수 없는 물건들입니다. 보안 검색에서 압수됩니다.

### 절대 금지
- ❌ **다리미/스팀다리미**: 화재 위험 (선내 세탁 서비스 이용)
- ❌ **양초/인센스**: 화재 위험
- ❌ **전기 코일 히터**: 화재 위험
- ❌ **서지 프로텍터 (접지형 멀티탭)**: 일부 선사 금지 — 일반 멀티탭은 OK
- ❌ **드론**: 대부분의 선사 금지
- ❌ **셀프 밸런싱 보드 (호버보드)**
- ❌ **무기류, 날카로운 물건**

### 선사별 다른 것
- 🍷 **주류**: 대부분 반입 금지 (일부 선사는 와인 1~2병 허용)
- 🥤 **음료**: 물, 탄산음료 반입 가능 여부는 선사별 상이
- 🔌 **연장 코드**: 접지 없는 멀티탭은 대부분 OK, 연장 코드는 금지

## 👔 드레스코드 가이드

크루즈에는 저녁 시간대 드레스코드가 있습니다. 선사마다 다르지만 일반적인 기준은:

### 캐주얼 (Casual)
- 깔끔한 일상복
- 청바지 OK, 반바지/슬리퍼 NO (디너 시)
- 대부분의 밤이 캐주얼

### 스마트 캐주얼 (Smart Casual)
- 남성: 카라 셔츠 + 긴 바지
- 여성: 원피스 또는 블라우스 + 스커트
- 일부 선사의 기본 디너 드레스코드

### 포멀 (Formal Night) 🎩
- 7박 크루즈 기준 1~2회
- 남성: 정장 또는 턱시도 (다크 수트도 OK)
- 여성: 이브닝 드레스 또는 칵테일 드레스
- **싫으면 패스 가능** — 뷔페나 캐주얼 레스토랑 이용

### 선사별 특징
| 선사 | 포멀 나이트 | 분위기 |
|------|-----------|--------|
| 로열캐리비안 | 1~2회 | 적당히 격식 |
| MSC | 1~2회 | 유럽풍 엘레강스 |
| NCL | 없음 | 완전 자유 (프리스타일) |
| 디즈니 | 해적 나이트 | 코스튬 파티 분위기 |
| Celebrity | 2~3회 | 세련된 격식 |

## 짐 싸기 꿀팁 🍯

1. **짐은 2개로 나누세요** — 큰 캐리어(부치는 짐) + 기내용 백팩
2. **승선 첫날 짐이 늦을 수 있음** — 수영복, 약, 귀중품은 기내용에
3. **압축팩 활용** — 옷 부피를 반으로 줄일 수 있음
4. **더러운 옷 전용 백** — 지퍼백이나 세탁망 준비
5. **짐 태그를 미리 출력** — 승선일 짐 위탁 시 필수
6. **필수품은 일찍 사세요** — 출발 전날 급하게 사면 빠뜨리기 쉬움
7. **객실 문 데코** — 자석 데코로 내 객실 쉽게 찾기 (같은 문이 수백 개!)
